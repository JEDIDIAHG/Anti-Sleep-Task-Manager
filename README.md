Jedidiah's Anti-Sleep Pro Task Manager 🚀

An interactive web-based reminder application designed to boost productivity and ensure users stay awake through cognitive engagement.

🌟 Features

Multi-Task Management: Add and track multiple reminders with unique timers.

Priority System: Categorize tasks by Low, Medium, or High priority with color-coded visual cues.

Data Persistence: Uses localStorage to ensure your tasks remain even after a page refresh or browser restart.

Anti-Sleep Logic: To stop an alarm, users must solve a random Multiplication Challenge (3x3 to 12x12).

Responsive UI: High-contrast gold and charcoal theme with CSS animations and full-screen alert overlays.

🛠️ Built With

HTML5: Semantic structure.

CSS3: Custom @keyframes animations and responsive layout.

JavaScript (Vanilla): DOM manipulation, state management, and real-time interval logic.

Developed as a practical application of Software Engineering and Management principles.
